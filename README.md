  <h1>Apartado 1: Crear cuenta en GitHub y crear "Hola Mundo"</h1>

  <p>- Accedemos a la web de Github y pulsamos “Sign up” para crear una nueva cuenta,  
  rellenamos los datos y pulsamos “Create Account”.  
  Una vez creada la cuenta, accedemos.</p>

  <p>- Para crear el repositorio pulsamos la opción “New”, junto a Top repositories,  
  le ponemos el nombre, descripción y otras opciones de configuración y pulsamos “Create repository” para finalizar.</p>

  <h1>Apartado 2: Conectar con la comunidad en redes sociales.</h1>

  <p>- Entramos en Twiter (https://x.com) y creamos la cuenta en caso de no tenerla y nos logueamos.  
  Para crear la lista, en el menú vertical de la izquierda, pulsamos “Lists”,  
  en la ventana de las listas, saldrán las creadas por nosotros y una sección de otras listas genéricas para conectarnos.  
  Para crear una nueva lista, pulsamos el icono “New List” en la esquina superior derecha,  
  rellenamos los campos de “Nombre”, “Descripción” y si queremos que sea, o no, privada,  
  y, en el campo de búsqueda que saldrá a continuación, buscaremos los perfiles que nos interese añadir a la lista y los añadiremos pulsando el botón “Add”.  
  Una vez finalizado, pulsamos “Done” para finalizar la creación de la lista.</p>
